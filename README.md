# 🌿 Nature Defenders v1.0 - Mobile Testing

**Text-free cooperative strategy game with hexagonal grid gameplay**

## 🎮 Play Now

Open `nature_defenders_v1.html` in your mobile browser for the complete experience.

## ✨ Features

- **Text-free gameplay** - All mechanics expressed through icons only
- **Hexagonal grid** - 10x10 responsive layout optimized for mobile screens  
- **Drag & drop** - Touch-friendly card placement with visual feedback
- **Feedback systems** - Particle effects, haptic vibration, and audio feedback
- **Responsive design** - Adapts to any screen size with fullscreen mode

## 🎯 How to Play

1. **Drag cards** from your hand onto the hexagonal grid
2. **Watch for effects** when cards interact with adjacent cards
3. **Score points** by successfully placing cards
4. **Use the legend** (📖 button) to learn card interactions

## 📱 Mobile Features

- **Haptic feedback** - Phone vibrates when you pick up/drop cards
- **Audio feedback** - Satisfying sounds for all interactions  
- **Fullscreen mode** - Tap ⛶ for immersive experience
- **Touch optimized** - Large touch targets and smooth animations

## 🎨 Card Types

### Growth & Nature
- 🌿 **Growth** - Spreads naturally
- 🐾 **Predator** - Controls other species  
- 🔥 **Fire** - Fire-resistant species
- 🛡️ **Defense** - Protected species

### Control Methods  
- 🔨 **Manual** - Physical removal
- ✂️ **Cutting** - Mechanical control
- 💧 **Water** - Moisture management
- ☠️ **Chemical** - Broad-spectrum treatment
